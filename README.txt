setup guide :) 
----‐--———--------
1. Select the skyridge .apk and when prompted, install as app  

2. Open the app, and use the 'binding of isaac' folder when prompted for game 

3.  Tap the settings icon in the top right corner to see the content settings 

4. Tap 'production keys' and go into the prod.keys folder, select the prodkeys file 

5. Tap 'title keys' and do the same, except selecting the title.key file 

6. done ok have fun bye
